SocialSphere is a social networking web application built with Django, HTML, CSS, and JavaScript. It allows users to interact, connect, and share content through posts in a personalized feed.
Functionalities-
1.User Authentication-
Signup- Users can create a new account with a username, email, and password.
Login/Logout- Secure login and logout functionality to manage user sessions.

2.Profile Management-
Users can update their account settings, including profile information.

3.Social Interactions-
Follow/Unfollow- Users can follow others to see their posts in the feed, and unfollow if they wish.
User Suggestions- Recommended users to follow based on activity.

4.Posts-
Create Post- Users can make text posts.
Like Post- Users can like posts to show engagement.
Feed Suggestions- Posts from followed users appear in the user’s personalized feed.

5.Interactive Feed-
Displays posts from followed users.
Supports engagement through likes and suggestions.

Technologies & Libraries-
Backend- Django
Frontend- HTML, Tailwind CSS, JavaScript
Database- SQLite
AJAX- For dynamic updates of likes, follow/unfollow, and feed content without reloading the page.
Pillow- Handles image uploads
